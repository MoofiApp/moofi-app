# MOOFI Finance (\$MOFI)

## Overview

Official frontend repo of [moofi.app](https://www.moofi.app). Moonriver yield farming optimizer.

## Env

Required env vars:

```
REACT_APP_NETWORK_ID = 1285 // Network ID of Moonriver
```
s
## To Run

```
yarn
yarn start
```

The app will be running at http://localhost:3000/
